InternUser connected a USB device to the Windows VM and copied Confidential_Report.txt to the USB drive. The user also opened the copied file from USB. Later, Project_Plan.txt was deleted and moved to Recycle Bin. Several applications were opened to generate application execution artifacts.
Windows Event Viewer was used to analyze user logon and logoff activity. Security logs were filtered using Event IDs 4624, 4634, 4647, and 4625. These logs helped identify successful login and logout activity of InternUser. System logs were also checked for USB/device connection events.
WinPrefetchView was used to analyze application execution history from Windows Prefetch files. It showed programs executed by the user, including Notepad, CMD, PowerShell, Calculator, Paint, Chrome/Edge, and File Explorer. The Last Run Time and Run Count helped reconstruct user activity.
